## 🔄 DI 函數的調用流程

### 範例：創建旅程的完整流程

```python
# 1. 用戶發送請求
POST /api/trips
Authorization: Bearer eyJ0eXAiOiJKV1QiLCJhbGc...
Body: {"title": "東京7日遊", ...}

# 2. FastAPI 路由處理
@router.post("", response_model=TripResponse)
def create_trip(
    payload: TripCreate,
    svc: TripService = Depends(get_trip_service),        # ← 觸發 DI
    current_user: dict = Depends(get_current_token)
):

# 3. FastAPI 調用 get_trip_service()
def get_trip_service() -> TripService:
    repository = get_trip_repository()                   # ← 調用 Repository DI
    return TripService(trip_repository=repository)

# 4. FastAPI 調用 get_trip_repository()
def get_trip_repository() -> TripRepository:
    return TripRepository()                              # ← 創建 Repository

# 5. TripRepository 初始化
class TripRepository:
    def __init__(self):
        self.supabase = get_supabase()                   # ← 獲取 DB 連接

# 6. get_supabase() 返回單例連接
@lru_cache(maxsize=1)
def get_supabase() -> Client:
    return create_client(...)                            # ← 只創建一次

# 7. 完整的依賴鏈已建立
# FastAPI 現在有了完整配置的 TripService

# 8. 調用業務邏輯
user_id = current_user.get("sub")
trip = svc.create_trip(payload.model_dump(), user_id)
    └── Service 調用 Repository
        └── Repository 調用 Supabase
            └── 資料寫入資料庫

# 9. 返回響應
return TripResponse(**trip)
```

  -d '{
    "email": "test@example.com",
    "username": "testuser",

### 3. 登入獲取 Token
```bash
curl -X POST http://localhost:8000/api/users/login \
  -H "Content-Type: application/json" \
  -d '{
    "email": "test@example.com",
    "password": "password123"
  }'

# 保存返回的 access_token
export TOKEN="返回的_access_token"
```

### 4. 創建旅程
```bash
curl -X POST http://localhost:8000/api/trips \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "測試旅程",
    "description": "這是一個測試",
    "start_date": "2024-06-01",
    "end_date": "2024-06-05"
  }'
```

### 5. 獲取旅程列表
```bash
curl -X GET "http://localhost:8000/api/trips?page=1&page_size=10" \
  -H "Authorization: Bearer $TOKEN"
```

## 📞 問題排查

### Q: 啟動後 API 無法訪問資料庫？
**A:** 檢查：
1. Supabase SQL 腳本是否已執行？
2. `config/.env.local` 的 SUPABASE_URL 和 KEY 是否正確？
3. 資料庫表格是否創建成功？（用 SQL 查詢檢查）

### Q: JWT Token 無效？
**A:** 檢查：
1. `JWT_SECRET` 是否已設置？
2. Token 是否過期？（預設 24 小時）
3. Token 格式是否正確？（Bearer TOKEN）

### Q: 權限錯誤（403）？
**A:** 檢查：
1. 嘗試訪問的是否是自己的旅程？
2. JWT Token 中的 `sub`（user_id）是否正確？
3. RLS 策略是否正確設置？
